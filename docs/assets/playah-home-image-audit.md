# PlayAh Home Image Audit

Audit date: 2026-06-11

Scope: audit only. No UI code changed, no component image replacement, no repo image downloads. Sources checked: PlayAh.vn and Facebook PlayAh Photos.

## Source Access Notes

- PlayAh.vn: accessible. Product/grid/media URLs were found from homepage HTML, product pages, and official PlayAh.vn WordPress media endpoints.
- Facebook PlayAh Photos (`https://www.facebook.com/PlayAh.Official/photos`): not usable for photo audit in this environment. The request returned a Facebook app/profile shell with Open Graph profile image metadata, but not a browsable photo grid or reliable individual photo URLs. Continue with PlayAh.vn official assets only.
- No random internet images or third-party brand images were used.

## Overall Recommendation

Use PlayAh.vn product-led square assets for Home v1. Most current assets are ecommerce marketplace banners with promo copy, return-policy badges, and shipping text, so they are strongest when cropped to the product side. For wide hero, avoid using these images full-frame unless the design allows controlled cropping/overlay. If a new clean hero shoot is possible, make that the final hero.

Best available hero candidate from existing official assets: `herfeel_ha_hero.jpeg`, cropped right/product + water texture, because it feels clean, premium, and product-led. Risk: it still contains text and only represents one gel/HA care product rather than broad PlayAh range.
